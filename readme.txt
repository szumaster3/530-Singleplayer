RuneScape 2009 Experience

An offline RuneScape experience emulating gameplay from the 2009 era.

---

Overview

- Includes a bundled JRE — no separate Java installation required.  
- Designed for offline play.  
- Ideal for preservation, testing, and custom single-player modifications.

---

Usage

1. Run `launch_server.bat` to start the server.  
2. Run `launch_client.bat` to start the client.

---

Project Structure

- `/game/` — Server, client, and related data files.  
- `/resources/` — Plugins.  
- `/jre/` — Embedded Java runtime.  
- `client.zip` — Legacy client archive.
